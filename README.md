# CervicePro 6.0 Complete Prototype

Includes:
- No-login operator fault reporting
- Equipment-ID lookup for 59 machines
- QR-compatible IDs (scanner integration can replace manual entry)
- Locked machine identity and editable hours
- Runnable / Unsure / Not Runnable status
- Photo/video selection
- Password-protected admin dashboard
- Open and completed work orders
- Repair Confidence, root cause, actual fix, parts/labor cost
- Monthly $20,000 maintenance budget dashboard
- Machine folders and repeat-repair warning
- Installable/offline PWA
- Optional Google Apps Script email/Sheet backend

This is a working local/static prototype, not a production-secure company system.
